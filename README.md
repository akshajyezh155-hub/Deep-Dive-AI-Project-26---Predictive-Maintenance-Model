# Deep Dive AI Project 2026 - Predictive Maintenance Model
This project is a predictive model that I have done for the Deep Dive AI Workshop in the summer of 2026. The project is a predictive model about predictive maintenance, or the task of mechanics to predict when a vehicle shall fail before it fails. This is not only used by mechanics for everyday cars, but also by those who work on professional-grade cars, like F1.
